Jobly

Background

Jobly is a web application for posting, viewing, and applying to job applications. It’s built with React on the frontend and Express on the backend.

Key Learnings

‣ Authentication with JWTs
‣ Hydration
‣ Writing parametized queries to protect against SQL injections

Challenges

One interesting thing I worked on during this project was implementing a live search feature to automatically filter results as the user types. This was a challenge because we wanted to make sure we weren't firing off an API call with every keystroke.

To solve this problem, we used a debounce function to make sure an API call only occured when a user had stopped typing in the search bar for at least 800ms.

Further Work

We needed to prop drill quite a bit to pass down the current logged in user’s info. One area for improvement is using the Context API to obviate the need for prop drilling.
